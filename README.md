# ACHR-Ubuntu
This is a script to automatically install Mikrotik Chr on Ubuntu versions 20.04, 22.04, and 24.04.

# Usage
```
sudo su
```
```
git clone https://github.com/safrinnetwork/ACHR-Ubuntu
```
```
cd ACHR-Ubuntu
```
```
chmod +x install.sh
```
```
sudo apt-get install dos2unix
```
```
dos2unix install.sh
```
```
bash install.sh
```
